## File Naming

* Snake-case for all files and folders (e.g., `my_file.py`, `my_folder`)

## Commit Messages

```
[optional issue id]: <description>

[optional body]

[optional footer(s)]
```

**Examples**:
```
* [`#19`] - Add the operator precedence examples
* [`#####`] - Move the image assets to a dedicated folder
```
